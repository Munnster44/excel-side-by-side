About

The Excel Side-by-Side Viewer was developed by Glen Carruthers for CanBoat / NautiSavoir to compare membership and registration data across Excel spreadsheets.

Usage Instructions

  1. Select two Excel files using File A and File B.
  2. Choose a Compare Column (e.g., Membership Number).
  3. Click Compare to highlight matches (green) and non-matches (red).
  4. Use Show Matches, Show Non-Matches, or Show All to filter results.
  5. Use the Search box to filter across all columns in both tables.
  6. Use Sync Scroll to scroll both tables together, or toggle off for independent scrolling.
  7. Click Download Report to export the comparison results.
